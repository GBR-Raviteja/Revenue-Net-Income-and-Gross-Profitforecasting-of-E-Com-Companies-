# Revenue-Net-Income-and-Gross-Profitforecasting-of-E-Com-Companies-
This Project forecasts revenue, net income, and gross profit for Amazon, Alibaba, and Walmart using ARIMA, SARIMA, DNN, GRU, and BiLSTM time series models. Results show SARIMA outperforms others on seasonal E-Commerce datasets. The study highlights model comparisons and suggests future improvements with larger datasets.
